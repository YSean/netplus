git add files 添加准备提交的文件{放到暂存区}
git commit -m "commit message" 提交
git status 同步
git diff 对比文件
git log 查看版本
git log --pretty=online
git reset 回退版本
git reflog 可以记录每次执行命令的版本号
git checkout -- files 相当于SVN 的 reverts
git rm 在版本库中删除文件
